# Layer 1: Foundation

Defines core patterns that all platforms and stackkits inherit.

## Modules

| Module | Description |
|--------|-------------|
| minimal | Bare minimum, no opinions, maximum flexibility |
| base | Sensible defaults for homelabs and small teams |
| extended | Enterprise patterns, compliance, high availability |

## What Foundation Defines

- Network patterns (naming, subnets, isolation)
- Security defaults (SSH, firewall, secrets)
- Observability patterns (logging, metrics, health checks)
- Lifecycle hooks (pre/post deploy, backup, restore)
