# Layer 2: Platform

Defines container runtime and orchestration patterns.

## Modules

| Module | Description |
|--------|-------------|
| docker | Single host, simple deployment |
| docker-swarm | Multi-host clustering with Docker native tools |
| kubernetes | Full orchestration for complex workloads |

## What Platform Defines

- Container runtime configuration
- Orchestration and scheduling
- Service discovery mechanism
- Volume and storage patterns
- Networking mode (host, bridge, overlay)
