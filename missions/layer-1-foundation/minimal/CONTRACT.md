# Minimal Foundation Contract

No opinions. Maximum flexibility. User decides everything.

## Network
No default network created.
User must explicitly configure all networking.

## Security
No hardening applied by default.
User is responsible for all security decisions.

## Observability
No logging or monitoring configured.
User adds observability as needed.

## Configuration
No defaults provided.
All values must be explicitly set.

## Use Case
Advanced users who want full control.
Custom setups that don't fit standard patterns.
Testing and experimentation.
