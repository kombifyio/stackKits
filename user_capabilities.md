# User Capabilities

## Value Proposition
StackKits enables anyone to deploy a complete self-hosted infrastructure with pre-validated configuration templates that prevent mistakes before deployment.

## Capabilities

### Data Safety
Your configuration is checked for errors before anything runs—catching mistakes like mismatched settings, missing passwords, or invalid server addresses automatically.

### Flexible Deployment Options
Choose from ready-to-use templates: run everything on one machine for simplicity, spread services across multiple servers for performance, or set up automatic failover for reliability.

### Operating System Freedom
Works seamlessly with your preferred Linux distribution—templates automatically adjust for Ubuntu or Debian without manual changes.

### Template Reusability
Core setup steps (user accounts, firewalls, monitoring) are defined once and reused across all deployment types, reducing duplicate work.

### Guided Configuration
Start with working examples that include all necessary settings—just update your server addresses and passwords, then deploy.

## Benefits

### Efficiency
Ready-made templates eliminate hours of manual setup and reduce deployment time from days to minutes.

### Ease of Use
Pre-validated configurations catch errors before they cause problems, making infrastructure deployment accessible to non-experts.

### Reliability
Automated consistency checks ensure all services receive compatible settings and required dependencies are properly configured.

## 3-Step Walkthrough

### Input
Copy a template file and customize three things: your server addresses, login credentials, and service passwords.

### Action
Run a validation check to verify everything is correct, then execute the deployment command which connects to your servers.

### Result
Within minutes, your complete infrastructure is running—containers started, networks configured, monitoring active—ready for immediate use.
